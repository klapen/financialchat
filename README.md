# Financial Chat

Simple chat using NodeJS, with [Socket.io](https://socket.io/)

# Before you start

Clone the repository
```
$ git clone https://github.com/klapen/financialchat.git
$ cd financialchat
```

Install dependencies
```
$ npm install
```

# Start server

```
$ npm start
```
